# ঐক্যশক্তি ফাউন্ডেশন-Oikyoshokti Foundation — Frontend

React frontend for the Foundation website.

## Setup

```bash
npm install
npm start
```

Make sure the backend is running at `https://foundation-backend-private.onrender.com`.

## Build

```bash
npm run build
```

## Admin Login

URL: `http://localhost:3000/admin/login`  
Email: `admin@foundationx.org`  
Password: `Admin@123456`

## Environment

Edit `.env` to change the API URL:
```
REACT_APP_API_URL=https://foundation-backend-private.onrender.com/api/v1
REACT_APP_MEDIA_URL=https://foundation-backend-private.onrender.com
```
